# Static Cloudflare Pages Site Template

This is a simple template for creating a static website to be hosted on Cloudflare Pages. It includes HTML, CSS, and JavaScript files ready to be customized for your needs.

## Features

- Responsive design that works on all devices
- Modern CSS using variables and flexbox/grid layouts
- Clean HTML5 structure
- Simple contact form template
- Basic JavaScript functionality

## Directory Structure

```
├── index.html          # Homepage
├── about.html          # About page
├── contact.html        # Contact page
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── main.js         # Main JavaScript file
├── images/             # Directory for your images
│   └── favicon.png     # Site favicon
├── .gitignore          # Git ignore file
└── README.md           # This file
```

## Getting Started

1. Clone this repository
2. Customize the HTML, CSS, and JavaScript files to match your needs
3. Replace placeholder content with your own content
4. Add your own images to the `images` directory
5. Deploy to Cloudflare Pages

## Deployment

Follow the instructions in the included DEPLOYMENT.md file to deploy your site to Cloudflare Pages.

## Customization

- Change colors by modifying the CSS variables in the `:root` selector in `css/styles.css`
- Update the navigation links in the HTML files
- Add your own content to each page
- Extend functionality by adding more JavaScript in `js/main.js`
